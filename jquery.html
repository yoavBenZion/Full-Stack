how to got all child nodes to get all ?
in jquery its very simple. filter () and find()
$( "ul" ).find( "li" ).filter( ":even" );

There are many special selectors in jQ that are unike and easier to work wuth then the selectors availabole in Css3.

Chaining

$("li").filter(":even").end().filter(":odd")
end() - finish the previous commend and run somthing else.
in this example, it does the 2 oposites commends on the same li in the same code line.

get() vs []

$("li")get() will get all the li elements as array.
same as $("li")[] .
